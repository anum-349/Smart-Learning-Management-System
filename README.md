# **📚 Learning Management System (LMS)**

A modern, scalable **Learning Management System** (LMS) designed for universities and institutes.
Built with **Next.js**, **React**, **Tailwind CSS**, and **Node.js**, this system enables administrators, instructors, and students to manage academic operations seamlessly.

---

## 🚀 **Features**

### **🛠 Admin Panel**

* Manage Departments
* Manage Instructors
* Manage Students
* Create & Manage Courses
* Assign Courses to Instructors
* Manage Enrollments
* Notifications Panel
* Global Settings

### **👨‍🏫 Instructor Panel**

* View Assigned Courses
* Manage Classes & Attendance
* Upload Course Material
* Grade Students
* Manage Profile & Availability

### **🎓 Student Panel**

* View Enrolled Courses
* Download Resources
* Track Attendance
* View Grades / Transcript
* Notifications & Upcoming Events

## 🧪 **Running the Project Locally**

### **1️⃣ Clone the Repository**

```bash
git clone https://github.com/anum-349/Learning-Management-System.git
cd Learning-Management-System
```

### **2️⃣ Install Dependencies**

#### Frontend

```bash
cd frontend
npm install
npm run dev
```

#### Backend

```bash
cd backend
npm install
nodemon index.js
```
